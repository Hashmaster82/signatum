# Шрифты для PDF экспорта

Эта папка предназначена для хранения шрифтов, используемых при генерации PDF отчетов.

## Рекомендуемый шрифт

Для правильного отображения кириллицы в PDF отчетах рекомендуется использовать шрифт **Chakra Petch**.

### Установка шрифта

1. Скачайте шрифт Chakra Petch с [Google Fonts](https://fonts.google.com/specimen/Chakra+Petch)
2. Поместите файл `ChakraPetch-Regular.ttf` в эту папку
3. Перезапустите приложение

### Альтернатива

Если шрифт не установлен, приложение будет использовать стандартный шрифт Arial, но могут возникнуть проблемы с отображением кириллических символов.
